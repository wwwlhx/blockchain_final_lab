## 4. 答辩记录建议

建议至少记录并截图以下结果：

- 合约部署成功
- 结构化资产登记成功
- 链上完整确权记录查询成功
- 资产历史追溯成功
- `verify:asset` 校验通过
- 重复登记拦截
- `transfer:asset` 成功
- `revoke:asset` 成功
- Revoked 资产再次转移失败
- 修改文件后文件校验失败
- 修改 metadata 后元数据校验失败
